## Requirements

## Installation

PageTabBarController is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "PageTabBarController"
```

via Carthage
```
github "MingLoan/PageTabBarController"
```
